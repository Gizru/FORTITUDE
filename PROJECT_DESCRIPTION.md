# Fortitude - Project Description

## Short Description (GitHub Repository Description)

**A comprehensive automated testing framework for 42 school projects that validates builds, code style, and functionality with detailed reporting.**

## Medium Description (For README/About Section)

Fortitude is an automated testing framework designed specifically for 42 school projects. It provides a unified solution for building, validating, and testing C projects with integrated norminette checking, comprehensive test suites, and automatic cleanup. The framework automatically detects project structures, runs norminette style checks, executes organized test suites (Part 1, Part 2, Bonus), and provides detailed color-coded reports. Designed to work seamlessly from any project directory, Fortitude ensures code quality and correctness while maintaining clean repositories.

## Long Description (For Documentation)

Fortitude is a powerful, cross-platform testing framework built specifically for 42 school's curriculum projects. It automates the entire validation workflow for C projects, combining build verification, code style checking, functional testing, and artifact cleanup into a single, easy-to-use tool.

### Key Features

**Automated Workflow:**
- Detects project structure and lists all files
- Builds projects with appropriate targets (including bonus functions)
- Runs norminette code style validation
- Executes comprehensive test suites organized by subject parts
- Automatically cleans all build artifacts

**Intelligent Project Detection:**
- Automatically identifies 42 project types (Libft, etc.)
- Finds and links to test suites dynamically
- Works from any directory without configuration
- Handles both mandatory and bonus functions gracefully

**Comprehensive Testing:**
- Tests organized by subject structure (Part 1, Part 2, Bonus)
- Detailed pass/fail reporting for each function
- Color-coded output for easy readability
- Summary statistics and error reporting

**Developer Experience:**
- Simple one-command installation
- Global PATH integration
- Works from any project directory
- Cross-platform support (Linux, macOS, Windows)
- Extensible architecture for new projects

### Use Cases

- **Students:** Quickly validate projects before submission
- **Peer Evaluators:** Standardized testing during evaluations
- **Instructors:** Automated project validation
- **Personal Projects:** Maintain code quality in 42-style C projects

### Technical Highlights

- Written in C following 42 coding standards
- Modular architecture with reusable test framework
- Automatic dependency detection and handling
- Memory-safe with proper cleanup
- Norm-compliant codebase

Fortitude streamlines the development and validation process for 42 projects, ensuring code quality, style compliance, and functional correctness through automated testing and reporting.
